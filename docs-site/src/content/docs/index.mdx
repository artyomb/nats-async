## What It Covers

<CardGrid stagger>
  <Card title="Explicit Lifecycle" icon="setting">
    Start the client inside an `Async` task, keep idle sessions alive with periodic pings,
    and close or drain when the task finishes.
  </Card>
  <Card title="Core NATS" icon="document">
    Publish, subscribe, queue groups, request/reply, headers, and binary-safe payloads
    use a compact Ruby API.
  </Card>
  <Card title="JetStream" icon="open-book">
    Manage streams and consumers, publish with acknowledgements, fetch pull-consumer
    messages, and ack messages explicitly.
  </Card>
  <Card title="Examples First" icon="right-arrow">
    The examples are the intended API shape. Keep them short and use them as the
    starting point for supported workflows.
  </Card>
</CardGrid>
